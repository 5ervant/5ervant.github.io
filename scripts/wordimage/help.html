A) Installation and Activation - top

If you already purchased and downloaded the WordImage plugin in a .zip format, you may now follow these instructions for installation:

  1. In your WordPress administrative "Dashboard", go to Plugins -> Add New.
  2. In that page, click the "Upload Plugin" link next to the Add Plugins heading.
  3. Then click "Choose File" and select the .zip file that you've purchased and click "Open".
  4. And then click "Install Now" and wait for a few seconds.
  5. If "Plugin installed successfully" shown, you can now activate it by pressing "Activate Plugin" in the page where you are.
  6. And congrats, the plugin is now activated!

After activating, you can now see the "WordImage" menu in your "Dashboard".

C) Manual Shortcode Usage - top

You can manually use the WordImage shortcode on your page's content where you want a WordImage generator to appear, i.e.:

	[wordimage style="text-shadow"]Default Image Text[/wordimage]

The above shortcode example consists of 3 parts:

  1. The [wordimage] shortcode tag for embedding a WordImage generator.
  2. The text-shadow style that'll going to use for your generator.
  3. And the "Default Image Text" as the initial inputted text for the image text.

For example, you'll going to use "all" the generator styles, you can either use this shortcode:

	[wordimage style="all"]WordImage[/wordimage]

Or neglect the style attribute as the generator style "all" is assigned by default, such this one:

	[wordimage]WordImage[/wordimage]

D) TrueType Fonts - top

To maximize the uniqueness of your WordImage generator, you can delete all the pre-installed fonts and install your desired fonts.



Uploading TrueType Font

To upload your own TrueType font file:

  1. Your TrueType font file must be in a .zip format.
  2. In your WordPress administrative "Dashboard", go to WordImage -> TrueType Fonts.
  3. Click "Add TrueType Font", a dialog will show up.
  4. "Choose File" or "Browse" and open your TrueType font (must be in a .zip format).
  5. Then click "Install Now" and wait for a few seconds until the uploading be completed.

Deleting TrueType Font

To delete an installed TrueType font file:

  1. Go to WordImage -> TrueType Fonts.
  2. Point your mouse on the font that you want to delete, there you'll see a "Delete" button appearing.
  3. Click that "Delete" and verify it by following a click to the confirmation's "OK" button.

E) Generator Styles - top

Here's the list of the styles that you can use for your WordImage generator:

Name Description Version Added

two-text Convert a two-text, into a single image. v1.0

Go to WordImage -> Generator Styles and check the style location's components if you want to edit their style functionalities.
